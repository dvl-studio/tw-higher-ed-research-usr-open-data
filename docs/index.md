# Taiwan Higher Education Research & USR Open Data Pipeline

這個專案使用 Python 整理臺灣高教研究補助、教學實踐研究、國科會計畫、USR 年報與永續獎資料。

## 專案亮點

- 自動擷取多個公開資料來源
- 整理為結構化 Excel 資料
- 建立資料品質報告
- 提供互動式 Dashboard

## 專案連結

- GitHub Repository
- Streamlit Dashboard
- Data Quality Report
- Dashboard

## 技術

Python, pandas, requests, BeautifulSoup, lxml, openpyxl, Streamlit, Plotly, GitHub
